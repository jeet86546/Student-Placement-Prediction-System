# Student Placement Prediction
## Project Overview

The Student Placement Prediction System is a machine learning project that predicts whether a student is likely to get placed based on academic performance, skills, and other relevant factors.

The project uses machine learning techniques to analyze historical student data and identify patterns that influence placement outcomes.


## Objective

The goal of this project is to:

Analyze factors affecting student placements

Perform Exploratory Data Analysis (EDA)

Build a machine learning model for prediction

Provide placement predictions based on student attributes

## Features Used

The model uses the following features:

Branch

CGPA

10th Percentage

12th Percentage

Number of Backlogs

Attendance Percentage

Projects Completed

Internships Completed

Coding Skill Rating

Communication Skill Rating

Aptitude Skill Rating

Hackathons Participated

Certifications Count

City Tier

Extracurricular Involvement

## Model Used

The model used in this project:

Random Forest Classifier

Reasons for choosing Random Forest:

Handles non-linear relationships well

Works well with mixed feature types

Robust against overfitting
